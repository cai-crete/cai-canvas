## 코드 변경 계획

### Change A — `buildCrossComposite` Canvas 사이즈 및 배경색 변경 (~line 1667)

```ts
// 변경 전
const cellSize = 512;
canvas.width = cellSize * 3;   // 1536
canvas.height = cellSize * 3;  // 1536
ctx.fillStyle = '#121212';

// 변경 후 (V298 A: 16:9 + Pure White)
const cellW = 512;
const cellH = 288; // 16:9 기준: 864 / 3 ≈ 288
canvas.width  = cellW * 3;   // 1536
canvas.height = cellH * 3;   // 864
ctx.fillStyle = '#FFFFFF';
```
변경 시 `entries` 루프 내부의 `ctx.drawImage()` 호출도 `cellW, cellH`를 사용하도록 업데이트합니다.

---

### Change B — ELEVATION SHEET 전용 렌더 브랜치 추가 (~line 3121)

현행 코드에서 `item.type === 'artboard'` 다음에 `else` 블록으로 일반 `<img>` 렌더링이 이루어집니다.
**ELEVATION SHEET** (type: `'sketch_generated'` AND label: `'ELEVATION SHEET'`)에 대해 별도 브랜치를 삽입하여, 합성 이미지 위에 CSS 3×3 Grid 오버레이를 추가합니다.

```tsx
// 변경 전 (line 3121 이후)
) : (
  /* V276 G-2: editVersions displaySrc */
  <img src={...} .../>
)}

// 변경 후 (V298 B: ELEVATION SHEET 전용 렌더)
) : item.label === 'ELEVATION SHEET' ? (
  /* V298 B: ELEVATION SHEET — 합성 이미지 + React CSS Grid 라벨 오버레이 */
  <div className="w-full h-full relative" style={{ background: '#FFFFFF' }}>
    {/* 합성 이미지 베이스 */}
    <img
      src={item.src || ''}
      alt="elevation-sheet"
      className="w-full h-full object-contain pointer-events-none"
      draggable={false}
    />
    {/* CSS 3×3 Grid 라벨 오버레이 */}
    <div
      className="absolute inset-0 grid pointer-events-none"
      style={{
        gridTemplateColumns: '1fr 1fr 1fr',
        gridTemplateRows: '1fr 1fr 1fr',
        gridTemplateAreas: '". rear ." "left top right" ". front ."',
      }}
    >
      {/* 라벨 컴포넌트 (공통 스타일) */}
      {[
        { area: 'rear',  label: 'REAR'  },
        { area: 'left',  label: 'LEFT'  },
        { area: 'top',   label: 'TOP'   },
        { area: 'right', label: 'RIGHT' },
        { area: 'front', label: 'FRONT' },
      ].map(({ area, label }) => (
        <div
          key={area}
          className="flex items-end justify-center"
          style={{
            gridArea: area,
            paddingBottom: `${12 / (canvasZoom / 100)}px`,  // 셀 하단에서 12px 위
          }}
        >
          <span
            className="font-mono uppercase tracking-widest text-white bg-black opacity-90"
            style={{
              fontSize: `${12 / (canvasZoom / 100)}px`,
              padding: `${3 / (canvasZoom / 100)}px`,  // 상하좌우 3px
              display: 'inline-block',
            }}
          >
            {label}
          </span>
        </div>
      ))}
    </div>
  </div>
) : (
  /* V276 G-2: 기존 일반 이미지 렌더 */
  <img src={...} .../>
)}
```
