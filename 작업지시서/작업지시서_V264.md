## Change 1 — 잠금 아트보드 선택 허용 (~line 816)

### 현재 코드
```ts
// handleCanvasPointerDown 내부
if (item.type === 'artboard' && item.isLocked) return false;
```
이 `return false`가 drag 판정 함수에서 클릭 자체를 차단하여 선택이 안 됨.

### 변경
잠금 아트보드도 `selectedItemIds`에 추가 허용.
이동(drag) 차단만 유지 — 선택 후 드래그 시작 시점에 isLocked 체크:

```ts
// 드래그 시작 조건에서만 isLocked 검사
if (item.type === 'artboard' && item.isLocked) {
  // 선택은 허용, 드래그만 차단
  setSelectedItemIds([item.id]);
  return; // 드래그 초기화 생략
}
```

## Change 7 — CHANGE VIEWPOINT 트리거 변경

### 7-1. SitePlanDiagram 컴포넌트 — 드롭 타겟 추가 (~line 161)

컴포넌트에 `onImageDrop` prop 추가:

```tsx
const SitePlanDiagram = ({
  angle, isAnalyzing, analysisStep, lens, sitePlanImage,
  onImageDrop,  // 신규
}: {
  ...
  onImageDrop?: (imageDataUrl: string) => void;
}) => {
```

다이어그램 컨테이너에 드래그 이벤트 핸들러:

```tsx
<div
  className="relative flex items-center justify-center ..."
  onDragOver={(e) => { e.preventDefault(); e.dataTransfer.dropEffect = 'copy'; }}
  onDrop={(e) => {
    e.preventDefault();
    // A: 파일시스템에서 드래그
    const file = e.dataTransfer.files?.[0];
    if (file && file.type.startsWith('image/')) {
      const reader = new FileReader();
      reader.onload = () => onImageDrop?.(reader.result as string);
      reader.readAsDataURL(file);
      return;
    }
    // B: 캔버스 아이템에서 드래그 (dataTransfer에 이미지 URL 저장)
    const imgSrc = e.dataTransfer.getData('text/image-src');
    if (imgSrc) onImageDrop?.(imgSrc);
  }}
>
```

드롭 시 시각 피드백: `isDragOver` 상태로 border 하이라이트.

### 7-2. 캔버스 아이템 — 드래그 소스 추가 (~lines 2640-2700)

upload/generated 타입 아이템에 `draggable` 속성 + dragstart 핸들러:

```tsx
// 캔버스 아이템 렌더링 시 (upload/generated 타입)
<div
  draggable={item.type === 'upload' || item.type === 'generated'}
  onDragStart={(e) => {
    if (item.type === 'upload' || item.type === 'generated') {
      e.dataTransfer.setData('text/image-src', item.src ?? '');
      e.dataTransfer.effectAllowed = 'copy';
    }
  }}
  ...
>
```
