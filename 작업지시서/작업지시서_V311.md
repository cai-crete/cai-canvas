# 작업지시서 V311 (+ 버튼 1회 사용 후 제거)

**작성일**: 2026-04-01
**버전**: V311
**상태**: 승인 대기 (수행 계획 제안)

---

## 1. 개요 및 목표
파생 이미지에서 `+ 버튼`을 클릭하여 스케치 편집 모드로 진입한 경우, 해당 객체 상단의 제어바에서 중복된 `+ 버튼`을 제거하여 UI를 간결하게 유지하고 중복 작업을 방지합니다.

## 2. 작업 상세 내용

### 🚩 1. upload 이미지 제어바의 + 버튼 조건부 렌더링
*   `App.tsx` 내의 아이템 제어바 렌더링 로직 중 `upload` 타입 블록을 수정합니다.
*   `item.motherId`가 존재하지 않는 경우(최초 업로드 이미지)에만 `+ 버튼`을 표시하도록 합니다.
*   `+ 버튼` 클릭으로 인해 `motherId`를 보유하게 된 파생 upload 이미지에서는 해당 버튼이 자동으로 사라지게 됩니다.

---

## 3. 진행 안내
기존 이미지를 편집 모드로 전환한 뒤에는 추가적인 '편집 탭' 생성이 불필요하므로 이를 숨기는 처리를 진행하겠습니다.
위 계획에 대해 승인해 주시면 즉시 반영하도록 하겠습니다.
