## 변경 내용

### 연결 로직 교체

**기존** (motherId 기반):
- `item.motherId` 있는 아이템만 → 부모와 직선 연결

**변경** (전체 체인):
- 카드 타입 4종 필터: `upload | artboard | generated | sketch_generated` (path, text 제외)
- `canvasItems` 배열 순서(생성 순서) 기준 연속 쌍 연결
- `cardItems[0] → cardItems[1] → cardItems[2] → ...`

### 베지어 경로 (cai-planners `getBezierPath` 동일 방식)

```ts
// from: cardItems[i],  to: cardItems[i+1]
const startX = from.x + from.width;
const startY = from.y + from.height / 2;
const endX   = to.x;
const endY   = to.y   + to.height   / 2;

// 수평 방향 베지어 제어점
const cpX = startX + (endX - startX) * 0.5;

const pathD = `M ${startX} ${startY} C ${cpX} ${startY} ${cpX} ${endY} ${endX} ${endY}`;
```

### 스타일 (Q3 — 테마별)

| 상태 | 라이트 | 다크 |
|------|--------|------|
| 기본 | `rgba(0,0,0,0.2)`, strokeWidth 1.5 | `rgba(255,255,255,0.2)`, strokeWidth 1.5 |
| 하이라이트 | `#000000`, strokeWidth 3 | `#ffffff`, strokeWidth 3 |

### 하이라이트 조건 (Q2 — 선택 아이템 기준)

연결선 양 끝 아이템 중 하나라도 `selectedItemIds`에 포함되면 강조:

```ts
const isHighlighted = selectedItemIds.includes(from.id) || selectedItemIds.includes(to.id);
```

## 변경 코드 (line ~2231~2251 전체 교체)

```tsx
{/* 3. Connection Lines — V271: Bezier chain */}
<svg
  key="connection-svg"
  className="absolute inset-0 w-full h-full"
  style={{ overflow: 'visible' }}  {/* display: none 제거 */}
>
  <g style={{ transform: baseTransform }}>
    {(() => {
      const cardItems = canvasItems.filter((i: CanvasItem) =>
        ['upload', 'artboard', 'generated', 'sketch_generated'].includes(i.type)
      );
      return cardItems.slice(0, -1).map((from: CanvasItem, idx: number) => {
        const to = cardItems[idx + 1];
        const startX = from.x + from.width;
        const startY = from.y + from.height / 2;
        const endX   = to.x;
        const endY   = to.y + to.height / 2;
        const cpX    = startX + (endX - startX) * 0.5;
        const pathD  = `M ${startX} ${startY} C ${cpX} ${startY} ${cpX} ${endY} ${endX} ${endY}`;

        const isHighlighted =
          selectedItemIds.includes(from.id) || selectedItemIds.includes(to.id);

        const stroke = isHighlighted
          ? (theme === 'dark' ? '#ffffff' : '#000000')
          : (theme === 'dark' ? 'rgba(255,255,255,0.2)' : 'rgba(0,0,0,0.2)');
        const strokeWidth = isHighlighted ? 3 : 1.5;

        return (
          <g key={`chain-${from.id}-${to.id}`}>
            <path
              d={pathD}
              fill="none"
              stroke={stroke}
              strokeWidth={strokeWidth}
              style={{ transition: 'stroke 0.3s ease, stroke-width 0.3s ease' }}
            />
            {isHighlighted && (
              <circle r="3" fill={theme === 'dark' ? '#ffffff' : '#000000'}>
                <animateMotion dur="2s" repeatCount="indefinite" path={pathD} />
              </circle>
            )}
          </g>
        );
      });
    })()}
  </g>
</svg>
```
