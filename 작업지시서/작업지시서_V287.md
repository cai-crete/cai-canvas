### D — SKETCH TO IMAGE 배치 + 연결선

**현행 배치** (`~line 2030–2039`):
```ts

let newY = sourceItem ? sourceItem.y : 0;
```
→ 결과 이미지가 소스의 우측에 생성됨.

**신규**: 포지션 로직은 기존과 동일(첫 결과 우측 / 이후 하단). `newItemWidth`/`newItemHeight` 사전 계산 추가 (D-2 연결선 방향 판단에 활용).

**현행 연결선** (`~line 2368–2373`):
```ts
const startX = mother.x + mother.width;          // 모체 우측
const startY = mother.y + mother.height / 2;     // 모체 중앙
const endX   = child.x;                           // 자식 좌측
const endY   = child.y + child.height / 2;       // 자식 중앙
const cpX    = startX + (endX - startX) * 0.5;
const pathD  = `M ... C ${cpX} ${startY} ${cpX} ${endY} ...`; // 수평 베지어
```
→ 자식이 아래에 배치된 경우 연결선이 어색함.

**신규**: child 위치에 따라 방향 자동 결정:
- child가 모체보다 아래 (`child.y > mother.y + mother.height / 2`): **수직 베지어** (중앙 하단 → 중앙 상단)
- child가 모체보다 우측: **수평 베지어** (현행 유지)
