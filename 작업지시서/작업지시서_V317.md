# 작업지시서 V317 (우측 패널 레이아웃 최적화 및 간격 조정)

**작성일**: 2026-04-01
**버전**: V317
**상태**: 검토 대기

---

## 1. 개요 및 목표
11개의 버튼 확장으로 인해 우측 기능 선택기 패널이 잘리는 현상(Overflow)을 해결하고, 특정 그룹 간의 간격을 9px에서 12px로 넓혀 가독성을 높입니다.

## 2. 주요 분석 내용
*   **원인**: 기능 선택기 컨테이너의 최대 높이가 `max-h-[500px]`로 고정되어 있어, 11개의 버튼과 간격을 모두 담지 못하고 하단 버튼(`PRINT`, `+ ADD TOOLS`)이 잘림.
*   **수정 사항**:
    1.  **높이 제한 확장**: `max-h-[500px]` $\rightarrow$ `max-h-[700px]` 또는 `80vh`로 변경.
    2.  **스크롤 허용**: `overflow-hidden` $\rightarrow$ `overflow-y-auto` (필요 시 스크롤).
    3.  **간격 확대 (12px)**:
        *   **PARAMETER**와 **SKETCH TO PLAN** 사이: **12px** (기본 6px + 추가 6px)
        *   **CHANGE VIEWPOINT**와 **PRINT** 사이: **12px** (기본 6px + 추가 6px)

## 3. 주요 업데이트 계획

### ✅ 1. 컨테이너 레이아웃 수정
*   `App.tsx`의 기능 선택기 래퍼(`div`)의 Tailwind 클래스를 수정합니다.
    *   `max-h-[500px]` $\rightarrow$ `max-h-[700px]`
    *   `overflow-hidden` $\rightarrow$ `overflow-y-auto custom-scrollbar`

### ✅ 2. 12px 간격(GAP) 로직 업데이트
*   `App.tsx`의 맵 루프 내에서 특정 아이템(`PARAMETER`, `CHANGE VIEWPOINT`)에 적용된 `marginBottom` 스타일을 수정합니다.
    *   `marginBottom: '3px'` $\rightarrow$ **`'6px'`** (기본 6px + 추가 6px = 12px)
