## 개요

**목표**: cai-canvas PRINT 패널 GENERATE 클릭 시 → cai-print 새 탭 연동 + 자동 생성

**채택 방안**: **Method 1 — window.open + postMessage 연동**

> 채택 이유:
> - cai-print를 그대로 활용 (중복 개발 없음)
> - postMessage는 cross-origin base64 이미지 전달 가능 (URL 방식 불가)
> - 새 탭에서 cai-print 전체 기능(다운로드 등) 그대로 사용 가능

## 현황 분석

### cai-print 현재 상태
- URL 파라미터 수신 코드 없음
- postMessage 리스너 없음
- `LibraryImage = { src: string, type: string }` 구조
- `selectedImages: string[]` (src 배열)
- 이미지 추가 패턴: `setImages(prev => [{ src, type: 'UPLOADED' }, ...prev])`

### 연동 흐름

```
cai-canvas                              cai-print
──────────                              ─────────
GENERATE 클릭
  → window.open(cai-print.vercel.app)
  → listen 'PRINT_APP_READY'            ← useEffect 마운트 시
                                           window.opener.postMessage('PRINT_APP_READY', '*')
  → postMessage('INIT_PRINT', data)
     { purpose, orientation,            → message 수신
       numPages, textInput,               setXxx(data)
       images: string[] }                 setShouldAutoGenerate(true)
                                          useEffect → handleGenerate() 자동 실행
```

### 2-C: postMessage 리스너 useEffect (컴포넌트 상단 useEffect 영역)

```ts
// V270: cai-canvas 연동 — INIT_PRINT 수신 후 자동 생성
useEffect(() => {
  // 1. 준비 신호 전송
  if (window.opener) {
    window.opener.postMessage('PRINT_APP_READY', '*');
  }

  // 2. INIT_PRINT 수신
  const handler = (e: MessageEvent) => {
    if (e.source !== window.opener) return; // opener 검증
    if (e.data?.type !== 'INIT_PRINT') return;

    const { purpose: p, orientation: o, numPages: n, textInput: t, images: imgs } = e.data;

    setPurpose(p as Purpose);
    setOrientation(o as Orientation);
    setNumPages(n);
    setTextInput(t);

    if (imgs && imgs.length > 0) {
      const libImages = (imgs as string[]).map(src => ({ src, type: 'UPLOADED' }));
      setImages(prev => [...libImages, ...prev]);
      setSelectedImages(imgs as string[]);
    }

    setShouldAutoGenerate(true); // state 반영 후 다음 렌더에서 generate 트리거
  };

  window.addEventListener('message', handler);
  return () => window.removeEventListener('message', handler);
}, []); // 마운트 시 1회
```
