### Q1. Iframe 통신 방식

**배경**: iframe 내 cai-print에 어떻게 명령을 전달하고 데이터를 받을지 결정

| 방안 | 방식 | 특징 | 제약 |
|------|------|------|------|
| **A** | URL 파라미터 (`?mode=bridge&purpose=panel&numPages=3&text=...`) | 가장 단순, 별도 이벤트 코드 불필요 | Base64 이미지 데이터 URL 전달 불가 (길이 제한) |
| **B** | postMessage 양방향 이벤트 (**권장**) | 이미지·진행 상태 실시간 전달 가능, 확장성 높음 | change-viewpoint + cai-print 양쪽 코드 추가 필요 |
| **C** | API 폴링 (change-viewpoint가 cai-print API 엔드포인트 감시) | 가장 안정적 | cai-print에 API 서버 기능 별도 구축 필요 (현재 없음) |

> **권장**: 방안 B

---

### Q2. cai-print 결과물 캡처 방식

**배경**: cai-print는 html2canvas로 DOM을 캡처하여 PDF/PNG를 생성함.
iframe이 완전히 숨겨지면(0×0) DOM 레이아웃이 없어 html2canvas가 정상 작동하지 않음.
→ 아래 3가지 해결 방안 중 선택 필요.

| 방안 | 방식 | 특징 |
|------|------|------|
| **α** | iframe을 `position:fixed; top:-9999px; left:-9999px; width:1190px; height:842px; opacity:0; pointer-events:none`으로 숨김 | 실제 렌더링 공간 확보 → html2canvas 정상 작동. 사용자 눈에는 안 보이지만 DOM 존재 |
| **β** | html2canvas 캡처 없이 cai-print의 `PageData[]` (텍스트 + 이미지 src) JSON 직접 전달 → change-viewpoint는 이를 캔버스 아이템으로 추가 | 렌더링 제약 없음. 단, 최종 PDF 품질 이미지가 아닌 원본 데이터 전달 |
| **γ** | cai-print가 생성 완료 신호(`PRINT_COMPLETE`)만 전달 → 사용자가 cai-print 탭(새 탭으로 열기 fallback)에서 수동 다운로드 | html2canvas 제약 없음. 완전 무인 아님 — 사용자 액션 필요 |

> **권장**: 방안 α (레이아웃 숨김, html2canvas 유지)

---

### Q3. change-viewpoint 결과 반영 방식

**배경**: cai-print가 결과물을 전달한 후, change-viewpoint에서 어떻게 처리할지

| 방안 | 방식 |
|------|------|
| **α** | 생성된 각 페이지 이미지를 캔버스 아이템(upload 타입)으로 자동 추가 |
| **β** | PDF/PNG 자동 다운로드 (브라우저 다운로드 트리거) |
| **γ** | 캔버스 아이템 추가 + 동시에 다운로드 (둘 다) |

> **권장**: 방안 α (캔버스 아이템 추가)

```ts
else if (selectedFunction === 'PRINT') {
  if (!printPrompt) {
    alert("Please enter the printing requirements.");
    return;
  }

  // 선택된 아이템 이미지 수집
  const selectedImages = selectedItemIds
    .map(id => canvasItems.find((i: CanvasItem) => i.id === id))
    .filter((item): item is CanvasItem => !!item?.src)
    .map(item => item.src!);

  // printTemplate → cai-print purpose 매핑 (V270 동일)
  const purposeMap: Record<string, string> = {
    'REPORT': 'report',
    'DRAWING & SPECIFICATION': 'drawing',
    'PANEL': 'panel',
    'VIDEO': 'video'
  };
  const printPurpose = purposeMap[printTemplate] || 'report';

  setIsGenerating(true);
  setAnalysisStep('PRINT 생성 중...');

  // 기존 iframe 있으면 제거
  if (printIframeRef.current) {
    document.body.removeChild(printIframeRef.current);
    printIframeRef.current = null;
  }

  // 숨김 iframe 생성 (방안 α: 레이아웃 확보)

  iframe.style.cssText = 'position:fixed;top:-9999px;left:-9999px;width:1190px;height:842px;opacity:0;pointer-events:none;border:none;';
  document.body.appendChild(iframe);
  printIframeRef.current = iframe;

  const cleanup = () => {
    window.removeEventListener('message', messageHandler);
    if (printIframeRef.current) {
      document.body.removeChild(printIframeRef.current);
      printIframeRef.current = null;
    }
    setIsGenerating(false);
    setAnalysisStep('');
  };

  // postMessage 수신 핸들러 (방안 B)
  const messageHandler = (e: MessageEvent) => {
    if (e.source !== iframe.contentWindow) return;

    if (e.data?.type === 'PRINT_APP_READY') {
      // 데이터 전송
      iframe.contentWindow?.postMessage({
        type: 'PRINT_START',
        purpose: printPurpose,
        orientation: 'landscape',
        numPages: printPages,
        textInput: printPrompt,
        images: selectedImages
      }, 'https://cai-print.vercel.app');
    }

    if (e.data?.type === 'PRINT_PROGRESS') {
      setAnalysisStep(e.data.message || 'PRINT 생성 중...');
    }

    if (e.data?.type === 'PRINT_COMPLETE') {
      // [Q3 방안에 따라 결과 반영]
      // 방안 α: 캔버스 아이템 추가
      // 방안 β: 다운로드
      cleanup();
    }

    if (e.data?.type === 'PRINT_ERROR') {
      console.error('[V275] PRINT error:', e.data.error);
      cleanup();
    }
  };

  window.addEventListener('message', messageHandler);

  // 타임아웃 (60초)
  setTimeout(() => {
    cleanup();
  }, 60000);
}
```

```ts
      setSelectedImages(imgs as string[]);

    setShouldAutoGenerate(true);
  };

  window.addEventListener('message', handler);
  return () => window.removeEventListener('message', handler);
}, []);
```
