#### D-1: navigateStack 함수 추가

```ts
// V278: sketch_generated 스택 탐색 (childArtboardIds 기반)
const navigateStack = (parentId: string, direction: number) => {
  setCanvasItems((prev: CanvasItem[]) => prev.map((i: CanvasItem) => {
    if (i.id !== parentId || !i.childArtboardIds) return i;
    const maxIdx = i.childArtboardIds.length;
    const newIdx = Math.max(0, Math.min(maxIdx, (i.activeChildIndex ?? 0) + direction));
    return { ...i, activeChildIndex: newIdx };
  }));
};
```

### Change F: 연결선 — motherId 기반 팬아웃 구조

**현재**: 모든 card item을 순서대로 체인 연결 (A → B → C → D)

**문제**: 동일 모체(A)에서 파생된 B, C가 있을 때 B→C가 아닌 A→B, A→C로 연결되어야 함.

**신규 로직** (~line 2274):

```ts
// V278: motherId 기반 팬아웃 연결
const cardItems = canvasItems.filter((i: CanvasItem) =>
  ['upload', 'artboard', 'generated', 'sketch_generated'].includes(i.type)
);

return cardItems
  .filter((item: CanvasItem) => !!item.motherId)
  .map((child: CanvasItem) => {
    const mother = cardItems.find((c: CanvasItem) => c.id === child.motherId);
    if (!mother) return null;

    const startX = mother.x + mother.width;
    const startY = mother.y + mother.height / 2;
    const endX   = child.x;
    const endY   = child.y + child.height / 2;
    const cpX    = startX + (endX - startX) * 0.5;
    const pathD  = `M ${startX} ${startY} C ${cpX} ${startY} ${cpX} ${endY} ${endX} ${endY}`;
    const isHighlighted =
      selectedItemIds.includes(mother.id) || selectedItemIds.includes(child.id);
    const stroke = isHighlighted
      ? (theme === 'dark' ? '#ffffff' : '#000000')
      : (theme === 'dark' ? 'rgba(255,255,255,0.2)' : 'rgba(0,0,0,0.2)');
    const strokeWidth = isHighlighted ? 3 : 1.5;

    return (
      <g key={`chain-${mother.id}-${child.id}`}>
        <path d={pathD} fill="none" stroke={stroke} strokeWidth={strokeWidth}
          style={{ transition: 'stroke 0.3s ease, stroke-width 0.3s ease' }} />
        {isHighlighted && (
          <circle r="3" fill={theme === 'dark' ? '#ffffff' : '#000000'}>
            <animateMotion dur="2s" repeatCount="indefinite" path={pathD} />
          </circle>
        )}
      </g>
    );
  })
  .filter(Boolean);
```

> 비고: motherId 없는 아이템(초기 upload/artboard)은 연결선 없음.
> 연결선이 필요한 경우 추가 요청 바람.
