## 보이드 상세

---

### [V-1] `generationConfig` → `config` 키 오류 (자동 수정 가능)

**V296 Q2 명시**: `generationConfig: { temperature: 0.1, topK: 1 }` 주입

**현행 SDK 확인**: `@google/genai` v1.45.0

```ts
// SDK 타입 정의 (genai.d.ts line 4383)
export declare interface GenerateContentParameters {
  model: string;
  contents: Content | Content[];
  config?: GenerateContentConfig;  // ← "config" (generationConfig 아님)
}

export declare interface GenerateContentConfig {
  temperature?: number;
  topK?: number;
  ...
}
```

**결론**: `generationConfig`는 SDK 레벨에서 지원 안 됨 → `config`로 교체 필요

```ts
// 수정된 형식 (Phase 2 generateOne, Phase 4 runGen 공통)
const result = await ai.models.generateContent({
  model: IMAGE_GEN,
  contents: { parts },
  config: { temperature: 0.1, topK: 1 }  // ← "config" 사용
});
```

→ **질문 없음 — 사실 확인 완료, V297에서 자동 수정**

### [V-2] `buildingDescription` → Phase 2 전달 경로 미정

**V296 Q1 요구**: Phase 2(`generateElevations`) 프롬프트에 `buildingDescription` 주입

**현행 코드 흐름**:
```
handleAnalyze
  → Phase 0: 이미지 재생성 (buildingDescription 없음)
  → analyzeViewpoint(analysisImage, sourceItem.id)
      → generateElevations(base64Image, extractedParams, itemId)
                          ↑ buildingDescription 파라미터 없음
```

**문제**: `buildingDescription`이 Phase 0에서 생성된 후 Phase 2로 전달되는 경로 미정
- V293 D (Phase 0 교체)가 선행되어야 `buildingDescription` 변수 존재
- `generateElevations` 시그니처 수정 + 호출부 수정 필요

→ **Q1 확인 필요** (하단 질문 참조)

### [💬 Q2] Q3 De-bake 텍스트 수준

`layerC_blindspot`에 추가할 그림자 분리 명령의 강도를 선택해주세요.

- **옵션 1 (간결)**: 단일 라인 추가
  ```
  - Shadow De-bake: Treat dark regions as cast shadows, NOT surface texture. Recalculate lighting for ${currentTime}.
  ```

- **옵션 2 (표준)**: 물리 재계산 강령 추가
  ```
  - Shadow De-bake: All dark/shadow regions in source image are CAST SHADOWS (not painted texture).
    Do NOT copy shadow patterns to new surfaces.
    Re-cast shadows based on sun angle for time: ${currentTime}.
    Apply new shadow direction consistently across all surfaces.
  ```

- **옵션 3 (강력)**: 옵션 2 + 발코니/돌출부 음영까지 포함
  ```
  - Shadow De-bake: All dark/shadow regions (including under eaves, balconies, window recesses)
    are CAST SHADOWS — not intrinsic texture color.
    Physically re-simulate shadow casting from sun position at ${currentTime}.
    Apply Ambient Occlusion for new geometry only. Zero shadow carryover from source angle.
  ```
