## 현행 vs 신규 label 체계

| 단계 | 현행 label | 신규 label |
|------|------------|------------|
| 초기 아트보드 | (없음 → SKETCH) | (없음 → SKETCH) ← 변경 없음 |
| SKETCH TO IMAGE 결과 이미지 | `GENERATED` | `GENERATED` ← 변경 없음 |
| GENERATED [+] 아트보드 | `ORIGINAL` | (없음) |
| SKETCH artboard [+] 아트보드 | `EDIT 01~` | (없음) |
| (구 EDIT) 아트보드 SKETCH TO IMAGE 소스 | `EDIT 01` 유지 | `SKETCH` |
| (구 EDIT) 아트보드 SKETCH TO IMAGE 결과 | `EDITED 01~` | `GENERATED` |
| CHANGE VIEWPOINT 분석 소스 | `ANALYZED` | `GENERATED / VIEW` |
| CHANGE VIEWPOINT 결과 이미지 | `VIEWPOINT 01~` | `VIEW 01~` |
