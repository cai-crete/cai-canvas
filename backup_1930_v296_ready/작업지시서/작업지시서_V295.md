### Change E 구현 계획 — `architecturalSheetImage` Cross 합성 생성

**변경 범위**: `generateElevations` 완료 후 Canvas API로 5개 이미지를 십자 배치로 합성

#### 합성 로직 (Canvas API)

```
5개 개별 이미지 (front, top, right, left, rear)
  ↓ Canvas API 합성
단일 Cross 이미지 (3×3 격자)

그리드 크기: max(모든 이미지 width) × max(모든 이미지 height) 기준
셀 위치:
  rear  → (col 1, row 0) : x=cellW,  y=0
  left  → (col 0, row 1) : x=0,       y=cellH
  top   → (col 1, row 1) : x=cellW,   y=cellH
  right → (col 2, row 1) : x=2*cellW, y=cellH
  front → (col 1, row 2) : x=cellW,   y=2*cellH
빈 셀(4개): 투명 (배경 #121212)
```

#### `generateElevations` 후처리 추가 (~line 1624 이후)

```ts
// V295 E: 5뷰 Cross 합성 이미지 생성 (Canvas API)
const buildCrossComposite = async (imgs: typeof newImages): Promise<string | null> => {
  return new Promise((resolve) => {
    const entries = [
      { key: 'rear',  col: 1, row: 0 },
      { key: 'left',  col: 0, row: 1 },
      { key: 'top',   col: 1, row: 1 },
      { key: 'right', col: 2, row: 1 },
      { key: 'front', col: 1, row: 2 },
    ];
    const cellSize = 512; // px per cell
    const canvas = document.createElement('canvas');
    canvas.width = cellSize * 3;
    canvas.height = cellSize * 3;
    const ctx = canvas.getContext('2d');
    if (!ctx) { resolve(null); return; }
    ctx.fillStyle = '#121212';
    ctx.fillRect(0, 0, canvas.width, canvas.height);

    let loaded = 0;
    for (const { key, col, row } of entries) {
      const src = (imgs as any)[key];
      if (!src) { if (++loaded === 5) resolve(canvas.toDataURL('image/png')); continue; }
      const img = new Image();
      img.onload = () => {
        ctx.drawImage(img, col * cellSize, row * cellSize, cellSize, cellSize);
        if (++loaded === 5) resolve(canvas.toDataURL('image/png'));
      };
      img.onerror = () => { if (++loaded === 5) resolve(canvas.toDataURL('image/png')); };
      img.src = src;
    }
  });
};

const sheetImage = await buildCrossComposite(newImages);
if (sheetImage) setArchitecturalSheetImage(sheetImage); // state 저장
```

### Change E 확정 — 자동 캔버스 아이템 생성 추가

**동작 흐름**:
```
[ANALYZE 클릭]
    ↓ Phase 0 묘사 호출
    ↓ analyzeViewpoint (Phase 1 BIM 추출)
    ↓ generateElevations (Phase 2: 5방향 입면도 병렬 생성)
    ↓ buildCrossComposite (Canvas API: 5개 → 1개 십자 합성)
    ↓ setCanvasItems:
        ① 소스 아이템(IMAGE / VIEW) parameters 업데이트
           → elevationImages, sitePlanImage, architecturalSheetImage 저장
        ② 새 'ELEVATION SHEET' 캔버스 아이템 자동 생성
           → 소스 아이템 바로 아래 배치
           → 연결선: 소스 하단 → 시트 상단 (수직)
```

**캔버스 배치**:
```
[IMAGE / VIEW 아이템]
        ↓ 연결선 (수직)
[ELEVATION SHEET 아이템]  ← 자동 생성
```

const buildCrossComposite = (imgs: typeof newImages): Promise<string | null> =>
  new Promise((resolve) => {
    const entries = [
      { key: 'rear',  col: 1, row: 0 },
      { key: 'left',  col: 0, row: 1 },
      { key: 'top',   col: 1, row: 1 },
      { key: 'right', col: 2, row: 1 },
      { key: 'front', col: 1, row: 2 },

    const canvas = document.createElement('canvas');
    canvas.width = cellSize * 3;

    ctx.fillStyle = '#121212';
    ctx.fillRect(0, 0, canvas.width, canvas.height);
    let loaded = 0;
    for (const { key, col, row } of entries) {
      const src = (imgs as any)[key];
      if (!src) { if (++loaded === 5) resolve(canvas.toDataURL('image/png')); continue; }
      const img = new Image();
      img.onload = () => {
        ctx.drawImage(img, col * cellSize, row * cellSize, cellSize, cellSize);
        if (++loaded === 5) resolve(canvas.toDataURL('image/png'));
      };
      img.onerror = () => { if (++loaded === 5) resolve(canvas.toDataURL('image/png')); };
      img.src = src;
    }
  });

const sheetImage = await buildCrossComposite(newImages);

if (itemId) {
  setCanvasItems(prev => {
    const sourceItem = prev.find(i => i.id === itemId);

    // ① 소스 아이템 parameters 업데이트
    const updatedItems = prev.map(item => {
      if (item.id === itemId && item.parameters) {
        return {
          ...item,
          parameters: {
            ...item.parameters,
            elevationImages: newImages,
            sitePlanImage: newImages.top,
            architecturalSheetImage: sheetImage ?? null  // V295 E-1: 방안 1
          }
        };
      }
      return item;
    });

    if (!sourceItem || !sheetImage) return updatedItems;

    // ② ELEVATION SHEET 캔버스 아이템 자동 생성 (소스 하단)
    const sheetItem: CanvasItem = {
      id: `elevation-sheet-${Date.now()}`,
      type: 'sketch_generated',
      src: sheetImage,
      x: sourceItem.x,
      y: sourceItem.y + sourceItem.height + 100,
      width: sourceItem.width,
      height: sourceItem.width, // 합성 이미지는 1:1 정사각형
      motherId: itemId,
      label: 'ELEVATION SHEET',
      parameters: {
        architecturalSheetImage: sheetImage,
        angleIndex: 0, altitudeIndex: 0, lensIndex: 0, timeIndex: 0
      }
    };

    return [...updatedItems, sheetItem];
  });
}
