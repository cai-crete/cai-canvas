```typescript
    setShowStrokePanel(null);
  }
}}
```

### 변경 6: 두께 패널 UI

**위치**: 펜/지우개 버튼을 `<div className="relative">` 로 감싸고,
버튼 아래 패널 삽입 (Search 버튼 확장 패턴 동일)

**패널 스타일**:
- 위치: `absolute left-[calc(100%+12px)] top-1/2 -translate-y-1/2`
- 컨테이너: `flex items-center h-8 px-0.5 gap-0.5 rounded-full`
- 배경: `bg-white/80 dark:bg-black/80 border border-black/10 dark:border-white/10 backdrop-blur-sm`
- 그림자: `0 4px 16px rgba(0,0,0,0.08)`
- 오른쪽 확장 애니메이션: `@keyframes slideInRight` (translateX + opacity)

**두께 옵션 버튼 (5개: 1, 2, 4, 6, 8)**:
- 컨테이너: `w-8 h-8 flex items-center justify-center rounded-full transition-all`
- 활성: `bg-black dark:bg-white` (기존 활성 버튼 스타일)
- 비활성: `hover:bg-black/5 dark:hover:bg-white/5`
- 내부 원 크기: `width: size px, height: size px` (선택된 두께값 = 표시 원 직경)
- 내부 원 색상: 활성 시 white(dark: black), 비활성 시 black(dark: white)

```tsx
{/* 펜 두께 패널 */}
{showStrokePanel === 'pen' && (
  <div
    className="absolute left-[calc(100%+12px)] top-1/2 -translate-y-1/2 flex items-center h-8 px-0.5 gap-0.5 bg-white/80 dark:bg-black/80 border border-black/10 dark:border-white/10 rounded-full backdrop-blur-sm pointer-events-auto"
    style={{ boxShadow: '0 4px 16px rgba(0,0,0,0.08)', animation: 'slideInRight 0.15s ease-out' }}
  >
    {[1, 2, 4, 6, 8].map(size => (
      <button
        key={size}
        onClick={() => setPenStrokeWidth(size)}
        className={`w-8 h-8 flex items-center justify-center rounded-full transition-all hover:cursor-pointer ${
          penStrokeWidth === size
            ? 'bg-black dark:bg-white'
            : 'hover:bg-black/5 dark:hover:bg-white/5'
        }`}
      >
        <div
          className="rounded-full"
          style={{
            width: `${size}px`,
            height: `${size}px`,
            backgroundColor: penStrokeWidth === size
              ? (theme === 'dark' ? 'black' : 'white')
              : (theme === 'dark' ? 'white' : 'black'),
          }}
        />
      </button>
    ))}
  </div>
)}
```

지우개 패널도 동일 구조, `eraserStrokeWidth` 사용.

### 변경 7: 커서 원 — SVG 오버레이에 추가

**위치**: drawing SVG `<g style={{ transform: baseTransform }}>` 내부, 기존 요소 뒤

```tsx
{/* V255: Tool cursor indicator */}
{(canvasMode === 'pen' || canvasMode === 'eraser') && (
  <circle
    cx={lastMousePos.x}
    cy={lastMousePos.y}
    r={canvasMode === 'pen'
      ? penStrokeWidth / 2
      : eraserStrokeWidth}
    fill={canvasMode === 'pen'
      ? (theme === 'dark' ? '#FFFFFF' : '#000000')
      : 'none'}
    stroke={theme === 'dark' ? '#FFFFFF' : '#000000'}
    strokeWidth={canvasMode === 'eraser' ? 1 / (canvasZoom / 100) : 0}
    pointerEvents="none"
    opacity={0.6}
  />
)}
```

- 펜 커서: 채움 검정(다크: 흰색), 반지름 = `penStrokeWidth / 2`
- 지우개 커서: 채움 없음, 검정 테두리, 반지름 = `eraserStrokeWidth`
- `strokeWidth`는 줌에 반비례하여 화면에서 항상 1px 유지

### 변경 8: 애니메이션 keyframe 추가

`index.css` 또는 App.tsx의 `<style>` 태그 또는 inline CSS 처리:

```css
@keyframes slideInRight {
  from { opacity: 0; transform: translateY(-50%) translateX(-8px); }
  to   { opacity: 1; transform: translateY(-50%) translateX(0px); }
}
```

> 위 애니메이션은 패널 div의 `top-1/2 -translate-y-1/2`와 충돌하지 않도록
> `transform-origin`을 `left center`로 설정.

## 비고

- `const ERASER_RADIUS = 10` 상수 제거 → `eraserStrokeWidth` state로 대체
- 기존 path의 `strokeWidth`는 저장된 값(`item.parameters.strokeWidth`) 사용 — 소급 적용 없음
- 커서 원은 `pointerEvents="none"` — 드로잉 입력에 영향 없음
- `index.css` 접근 불가 시 App.tsx 내 `<style>` 태그 또는 Tailwind 커스텀 클래스 처리
