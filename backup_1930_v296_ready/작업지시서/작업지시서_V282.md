### Change B+C: CHANGE VIEWPOINT 로더 + Z-index 구조 교정

#### 배경

현재 로더는 캔버스 아이템 div 내부에 `z-[103]`으로 렌더링됨.
캔버스 아이템 div는 `transform`으로 인해 자체 stacking context를 형성하며,
z-[103]은 이 **로컬** stacking context 내부의 z값 → 캔버스 전역 레이어(z-[100]) 위로 올라오지 못함.

결과: 스케치 선 레이어(z-[100], 캔버스 전역)가 로더보다 시각적으로 위에 위치.

#### 해결 방안: 캔버스 레벨 로더 오버레이

로더를 캔버스 아이템 루프 바깥으로 분리하고, 캔버스 items transform wrapper와 동일한 좌표계를 공유하는 독립 오버레이로 렌더링.

**z-index 계층 (확정)**:
```
view controls (파란 테두리, 선택 UI): z-[105]
캔버스 레벨 로더 오버레이: z-[102]   ← 신규
스케치/연결선/Lasso 통합 오버레이:  z-[100]
캔버스 아이템 (selected):            z-20
캔버스 아이템 (normal):              z-10
```

#### B+C-2: 캔버스 레벨 로더 오버레이 삽입

`~line 2625` (토스트 알림 바로 아래, canvas items transform wrapper 바로 위) 삽입:

```tsx
{/* V282 B+C: 캔버스 레벨 로더 오버레이 (z-102) — 스케치 위(z-100), 뷰 컨트롤 아래(z-105) */}
{(isAnalyzing || isGenerating) && selectedItemIds.length > 0 && (
  <div
    className="absolute inset-0 pointer-events-none z-[102]"
    style={{
      transform: `translate(calc(50% + ${canvasOffset.x}px), calc(50% + ${canvasOffset.y}px)) scale(${canvasZoom / 100})`,
      transformOrigin: '0 0'
    }}
  >
    {selectedItemIds.map((itemId: string) => {
      const loadingItem = canvasItems.find((i: CanvasItem) => i.id === itemId);
      if (!loadingItem || loadingItem.type === 'path' || loadingItem.type === 'text') return null;
      return (
        <div
          key={`loader-${itemId}`}
          style={{
            position: 'absolute',
            left: loadingItem.x,
            top: loadingItem.y,
            width: loadingItem.width,
            height: loadingItem.height,
          }}
          className="flex items-center justify-center bg-white/50 dark:bg-black/50 backdrop-blur-md"
        >
          <Loader2 size={42} className="animate-spin text-white" />
        </div>
      );
    })}
  </div>
)}
```

**효과**:
- `isAnalyzing`: CHANGE VIEWPOINT 분석 중 원본 이미지 위에 로더 표시 ✓
- `isGenerating`: SKETCH TO IMAGE / CHANGE VIEWPOINT 생성 중 선택 아이템 위 로더 표시 ✓
- z-[102]: 스케치 선 레이어(z-100)보다 위에 위치 → Q2 답변 충족 ✓

```
ANALYZED (upload 또는 artboard 타입)  ← Book 버튼 위치
  └─ VIEWPOINT 01 (generated 타입)    ← Book 버튼 없음
```
